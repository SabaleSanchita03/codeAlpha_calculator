# Advanced Calculator 🧮

A responsive calculator built using HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations (+, -, ×, ÷)
- Dark/Light mode
- Keyboard support
- Calculation history
- Responsive UI
- LocalStorage support
